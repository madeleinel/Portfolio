    • Let's Paint

    A drawing application which enables the user to draw on a canvas and customise features such as the background colour, brush effects and the brush shape, as well as undoing and redoing drawing actions.

    Tech Stack: HTML5 Canvas, HTML5, CSS3, ES6

    Website Source code

    How it works:

    It was inspired by Wes Bos’s JavaScript30 HTML5 canvas tutorial. The canvas and the basic drawing feature was set up using this tutorial, and all additional features have been added using ES6.

    More features are in the process of being added.

    Tech stack:

    ¬ HTML5 Canvas to set up the canvas

    ¬ ES6 to enable the drawing features

    ¬ HTML5 and CSS3 to style the site

    Challenge One:

    Enable the user to undo each drawing action:

    Solution:

    When start each new drawing action, the system saves the last version of the site to an array. When clicking the Undo button, the canvas is restored to the top item of the array, and that item is then removed from that array to enable the user to undo all drawing actions one by one.
