    • Women To Tech

    A platform that matches women in tech with mentors in the industry.

    The website was created as part of the Code First: Girls Advanced Python Web Development Course competition with E Keane.

    How it works:

    Prospective mentors and mentees sign up to the platform by tweeting us using specific tags.

    When they sign up, a profile is created for them on the website, and they are sent their top three matches based on the information they have provided.

    Tech stack:

    ¬ HTML5 and CSS3 for front-end

    ¬ Python 2 for backend

    ¬ PostgreSQL for database handling

    ¬ Twitter API for Twitterbot
